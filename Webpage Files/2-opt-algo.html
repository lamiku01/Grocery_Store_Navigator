Grocery Store Navigator - 2-Opt Algorithm

Description

The Grocery Store Navigator uses the simple search algorithm 2-opt for it's calculations. This algorithm attempts to find the shortest path between a given list of nodes. In general, the algorithm works by swapping two arcs in the hope that is results in a shorter overall path distance. If the swap does result in a shorter path, the current path is updated. This process continues until there are no more improvements or all combinations of arc swaps have been completed.

Implementation

This implementation of the 2-opt algorithm keeps the first and last nodes on the path constant. The first node will always be the Entrance, and the last node will always be the Exit. This idea better reprents the decision making and path finding any given person would use when shopping at a grocery store. They must begin the trip by walking into the entrance, and eventually leaving through the exit.
