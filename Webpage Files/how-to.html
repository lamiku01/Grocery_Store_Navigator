Grocery Store Navigator - How To Use

General Overview

The Grocery Store Navigator is meant to be a simplistic solution to mapping out a rough path through the store given your grocery list. As a result, the tool simplifies the search by pre-selecting a few qualities. First, when the user searches for an item, only the first instance of the item in the API will be returend. Second, all items searched will be restricted to the pre-defined store location only. Additonally, some items are only listed with a specific aisle location in the API. These items instead are located by catergory. The category space and size can vary, so the exact item location is not as precise as if it had an aisle location.

Instructions

  • Enter the items separated by a comma and a space. Incorrect formatting will result in an error-message pop-up.
  • Only use generic item names. Do not use brand-names or specific item titles. For example: write "ice cream" instead of "Ben And Jerry's Chocolate Ice Cream
  • Click the "Submit" button once. Selecting the button again before the process is complete may result in extraneous results.
